# Images Directory

Folder ini berisi gambar-gambar yang digunakan dalam aplikasi Warta.

## Struktur Folder

- `hero/` - Gambar untuk hero section dan landing page
- `icons/` - Icon dan ilustrasi kecil
- `illustrations/` - Ilustrasi dan gambar pendukung

## Sumber Gambar yang Direkomendasikan

Untuk mendapatkan gambar berkualitas tinggi yang relevan dengan tema laporan dan komunitas, silakan download dari:

1. **Unsplash** (https://unsplash.com)
   - Search: "community service", "urban issues", "report management", "city problems"
   - Free untuk penggunaan komersial

2. **Pexels** (https://pexels.com)
   - Search: "community", "urban", "report", "document"
   - Free untuk penggunaan komersial

3. **Freepik** (https://freepik.com)
   - Search: "report illustration", "community service", "urban"
   - Perhatikan lisensi (beberapa memerlukan atribusi)

## Gambar yang Diperlukan

1. Hero Section:
   - Gambar komunitas/urban yang relevan
   - Ukuran: 1920x1080 atau lebih besar
   - Format: JPG/PNG

2. Icons/Illustrations:
   - Icon untuk dashboard cards
   - Ilustrasi untuk empty states
   - Format: SVG/PNG

## Catatan

Gambar-gambar ini akan digunakan di berbagai halaman untuk meningkatkan visual appeal dan user experience.
